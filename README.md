# PicBandApp

App para fins de contratar um músico para o seu evento. O App conciste em, escolher uma banda já pronta, criar uma banda através de uma seleção de músicos de vários generos musicais e contratar apenas um cantor(a) para eventos Pockets.

Existem duas áreas basicamente.

1 - Área do contratante
2 - Área do músico
